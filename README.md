# CI/CD MikroTik Edge IoT

Repository ini berisi implementasi CI/CD untuk otomasi konfigurasi jaringan MikroTik
pada lingkungan Edge IoT menggunakan Ansible dan GitHub Actions.

## Arsitektur
- GitHub Actions sebagai pipeline CI/CD
- Ansible sebagai automation engine
- MikroTik sebagai target konfigurasi
- Edge IoT sebagai client jaringan

## Fitur
- SSH key-based authentication
- Otomasi konfigurasi MikroTik
- Simulasi perubahan IP Edge IoT
